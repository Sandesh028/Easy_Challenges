PCAP (Tshark/Wireshark) Challenges

1) HTTP Credentials
Task: Analyze the provided HTTP packet capture file(s) to locate credentials or other cleartext data that contain a flag. When you find the flag, save it as plain text in a file named FLAG.txt inside this directory.

2) DNS Exfiltration
Task: Analyze the provided DNS packet capture file(s) to find suspicious DNS queries that encode data (a flag). When you find the flag, save it as plain text in a file named FLAG.txt inside this directory.
